Description

Die Regel von l'Hospital ist ein fundamentales Theorem der Analysis zur Bestimmung von Grenzwerten von Quotienten von Funktionen.

Theorem Statement

Natural Language Formulation:

Seien a, b ∈ ℝ ∪ {-∞, ∞}, f, g: (a,b) → ℝ differenzierbar mit g'(x) ≠ 0 für alle x ∈ (a,b). Sei x₀ ∈ {a,b} und es gelte (a) lim_{x→x₀} f(x) = 0 = lim_{x→x₀} g(x) oder (b) lim_{x→x₀} f(x) = ∞ = lim_{x→x₀} g(x). Falls lim_{x→x₀} f'(x)/g'(x) existiert, dann gilt lim_{x→x₀} f(x)/g(x) = lim_{x→x₀} f'(x)/g'(x).

Formal Notation (HOL/Why3):

∀(a b : ℝ) (f g : ℝ → ℝ) (x₀ : ℝ), (a < b) ∧ (∀x ∈ (a,b), ∃f'(x), ∃g'(x), g'(x) ≠ 0) ∧ ((lim_{x→x₀} f(x) = 0 ∧ lim_{x→x₀} g(x) = 0) ∨ (lim_{x→x₀} f(x) = ∞ ∧ lim_{x→x₀} g(x) = ∞)) ∧ (∃ lim_{x→x₀} f'(x)/g'(x)) → lim_{x→x₀} f(x)/g(x) = lim_{x→x₀} f'(x)/g'(x)

Proof (6 steps)

Proof Method: structured

1 Vorbereitung: Stetige Fortsetzung
Da f, g auf (a,b) differenzierbar sind, sind sie auch stetig. Wegen lim_{x→a} f(x) = 0 = lim_{x→a} g(x) können f und g stetig auf [a,b) fortgesetzt werden mit f(a) = 0 = g(a).
∀ε > 0, ∃δ > 0 : ∀x ∈ (a, a+δ), |f(x)| < ε ∧ |g(x)| < ε

2 Anwendung des verallgemeinerten Mittelwertsatzes
Sei x ∈ (a,b). Die Funktionen f und g sind auf [a,x] stetig und auf (a,x) differenzierbar. Mit g'(y) ≠ 0 für alle y ∈ (a,x) liefert der verallgemeinerte Mittelwertsatz: Es existiert ξ ∈ (a,x) mit f(x)/g(x) = [f(x) - f(a)]/[g(x) - g(a)] = f'(ξ)/g'(ξ).
∃ξ ∈ (a,x), f(x)/g(x) = f'(ξ)/g'(ξ)
Referenzen:

3 Betrachtung einer Folge gegen x₀
Sei (x_n)_{n∈ℕ} eine Folge mit lim_{n→∞} x_n = a und x_n ∈ (a,b) für alle n. Dann existieren ξ_n ∈ (a, x_n) mit f(x_n)/g(x_n) = f'(ξ_n)/g'(ξ_n) für jedes n ∈ ℕ.
∀n ∈ ℕ, ∃ξ_n ∈ (a, x_n), f(x_n)/g(x_n) = f'(ξ_n)/g'(ξ_n)
Referenzen:
theorem_generalized_mvt
4 Anwendung der Einschließungsregel (Squeeze-Theorem)
Wegen a < ξ_n ≤ x_n → a folgt aus der Einschließungsregel: lim_{n→∞} ξ_n = a.
lim_{n→∞} ξ_n = a
Referenzen:

5 Grenzwertberechnung durch Komposition
Da lim_{n→∞} ξ_n = a und ξ_n ∈ (a,b), und da lim_{x→a} f'(x)/g'(x) existiert, folgt durch Komposition von Grenzwerten: lim_{x→a} f(x)/g(x) = lim_{n→∞} f'(ξ_n)/g'(ξ_n) = lim_{x→a} f'(x)/g'(x).
lim_{x→a} f(x)/g(x) = lim_{x→a} f'(x)/g'(x)
Referenzen:

6 Schlussfolgerung
Dies ist genau die zu beweisende Aussage.
lim_{x→x₀} f(x)/g(x) = lim_{x→x₀} f'(x)/g'(x) ∎
